Getting started

Up and running in three steps

Start Ollama

Make sure Ollama is running with at least one chat model pulled, e.g. ollama pull qwen3.6:35b-a3b-q4_K_M

Start Replica

From the replica folder run node server.js — no installs, no build step, no dependencies.

Build

Open localhost:4747/agent, say what you want to make, and watch it happen.

What are you waiting for?

Your models are loaded. Your disk is ready. The Agent is listening.
