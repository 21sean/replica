📖 How Replica works

Everything is local. The Agent is whatever Ollama model you pick. Projects are plain folders in replica/projects/ on your disk.

Building. Describe what you want on the Home screen. The Agent plans, then writes real files — you'll see each one land with a ✓. The preview refreshes automatically.

Iterating. Keep chatting in the project: "make the header sticky", "add dark mode", "fix the bug where…". The Agent sees your current files each turn — including any manual edits you made.

Code tab. Full editor over the project files. Ctrl+S saves. Your edits become part of the Agent's context.

Console tab. Run node file.js or python file.py inside the project folder (60s limit).

Models. Bigger models build better apps. Thinking models (qwen3.6, gemma4) show their reasoning — collapse it with a click. First response after idle can take a while as the model loads into memory.
